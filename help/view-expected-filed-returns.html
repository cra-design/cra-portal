CRA account help - Businesses

Viewing expected and filed returns – CRA account help - Businesses

You can view a list of expected returns and their due dates as well as detailed information about your filed returns in My Business Account.

Expected returns

The expected returns table lists the returns we are expecting, and their respective due dates.

The Status column will display one of the following:

  • Return overdue: The return is late
  • Expected return: The return is expected but is not overdue
  • Next return: This return has an upcoming due date

Filed returns

This filed returns tables displays information for filed returns from the previous six years. If a return has been reassessed, the most recent reassessment will be displayed first.

The Status column will display one of the following:

  • Received: We received your return, but have not processed it yet
  • Assessed: We processed your return, and a Notice of Assessment has been mailed
  • Processed - No notice: We processed your return, and a Notice of Assessment was not mailed
  • Adjustment requested: We received your adjustment request, but have not processed it yet
  • Reassessed: We changed your assessment, and a Notice of Reassessment has been mailed
  • Adjusted - No notice: We processed your adjustment request, and a Notice of Assessment was not mailed
  • Adjustment denied: We denied your adjustment request

You can view detailed line-by-line information of a GST34, GST494, RC7200 or RC7294 return by selecting the hyperlink in the Reporting Period column.

Information returns are not included

The Filed returns table does not display the following information returns:

  • Form GST106, Information on Claims Paid or Credited for Foreign Conventions
  • Form GST111, Financial Institution GST/HST Annual Information Return
