The "File a return" service within your CRA business account allows you to electronically file 3 types of GST/HST returns by selecting one of the following radio button options:

  • Form GST34, Goods and services tax/harmonized sales tax (GST/HST) return for registrants
  • Form GST106, Information on claims paid or credited for foreign conventions and tour packages
  • Form RC7200, Goods and services tax/harmonized sales tax (GST/HST) and Quebec sales tax (QST) return for selected listed financial institutions

On this page

  • When to select GST34
  • When to select GST106
  • When to select RC7200

When to select GST34

Selecting Form GST34, Goods and services tax/harmonized sales tax (GST/HST) return for registrants allows you to file your GST34 return along with any applicable schedules and rebates.

Using this option, you can also file Form GST106,â€¯Information on claims paid or credited for foreign conventions and tour packages, along with your return.

When to select GST106

Select Form GST106, Information on claims paid or credited for foreign conventions and tour packages if you do not want to file Forms GST106 and GST34 at the same time.

If you want to file both forms at the same time, select the GST34 option instead.

If you paid or credited rebate amounts for foreign conventions or tour packages, and you claimed a deduction on your GST/HST return for the amount of GST/HST you paid or credited, then you must file this form. File this return for each reporting period in which you paid or credited these amounts and claimed a deduction on your GST/HST return.

When to select RC7200

Form RC7200, Goods and services tax/harmonized sales tax (GST/HST) and Quebec sales tax (QST) return for selected listed financial institutions is a combined return that is used by registrants that are selected listed financial institutions (SLFIs) with a permanent establishment in Quebec. These include listed financial institutions that would only be an SLFI for GST/HST purposes if Quebec were a participating province, and who have monthly or quarterly reporting requirements. For the definitions of SLFI for GST/HST purposes and SLFI for QST purposes, see Guide RC7250, GST/HST and QST Information for Selected Listed Financial Institutions.

Do not use this form if you are an SLFI for GST/HST purposes that does not have a permanent establishment in Quebec. Instead, use Form GST34, Goods and Services Tax/Harmonized Sales Tax (GST/HST) Return for Registrants.
