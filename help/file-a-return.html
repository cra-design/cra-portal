CRA account help - Businesses

Which GST/HST (and QST) returns to file – CRA account help - Businesses

You may be looking for:

  • File your GST/HST return
  • Instructions for preparing a GST/HST return
 

The "File a return" web form in My Business Account allows you to choose from 3 types of GST/HST and Quebec sales tax (QST) returns to file. The GST34 option allows you file both GST34 and GST106 returns at the same time. The RC7200 option allows you to file GST/HST and QST returns together.

On this page

  • When to select GST34
  • When to select GST106
  • When to select RC7200

When to select GST34

Selecting Form GST34, Goods and services tax/harmonized sales tax (GST/HST) return for registrants allows you to file your GST34 return along with any applicable schedules and rebates.

Using this option, you can also file Form GST106, Information on claims paid or credited for foreign conventions, along with your return.

For information on how to enter your information, refer to Completing a GST/HST return in My Business Account.

When to select GST106

Select Form GST106, Information on claims paid or credited for foreign conventions if you do not want to file Forms GST106 and GST34 at the same time.

If you want to file both forms at the same time, select the GST34 option instead.

If you paid or credited rebate amounts for foreign conventions, and you claimed a deduction on your GST/HST return for the amount of GST/HST you paid or credited, then you must file this form. File this return for each reporting period in which you paid or credited these amounts and claimed a deduction on your GST/HST return.

For more information, refer to Completing a GST106 return in My Business Account.

When to select RC7200

Form RC7200, Goods and services tax/harmonized sales tax (GST/HST) and Quebec sales tax (QST) return for selected listed financial institutions is a combined return that is used by registrants that are selected listed financial institutions (SLFIs) with a permanent establishment in Quebec. These include listed financial institutions (LFIs) that would only be an SLFI for GST/HST purposes if Quebec were a participating province, and who have monthly or quarterly reporting requirements.

Do not use this form if you are an SLFI for GST/HST purposes that does not have a permanent establishment in Quebec. Instead, use Form GST34, Goods and Services Tax/Harmonized Sales Tax (GST/HST) Return for Registrants.
