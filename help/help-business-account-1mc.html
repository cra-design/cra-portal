Message Centre Help

1. Which messages will be listed in the Message Centre?

The Message Centre will list all available messages that the business owner or authorized representative is authorized to view. A business owner and an authorized representative who is authorized for all accounts can view the messages for all accounts. An authorized representative who is authorized only for specific types of accounts can only view the messages for those types of accounts (RC, RP, RT, RD, etc.).

2. What information will be displayed?

The following information will be displayed for each message:

  • Date sent
  • A subject line containing a brief description of the message purpose and a link to the message content

3. Can a message be saved?

No. A message will be held in the Message Centre until it expires or is deleted.

4. Can a message be deleted?

Yes. There are three options for deleting messages:

  • Delete one message by selecting the checkbox next to the message, and then select the Delete button.
  • Delete multiple messages at the same time by selecting the checkbox next to each message, and then select the Delete button.
  • Delete all messages at the same time by selecting the top left corner checkbox, and then select the Delete button.

5. Where did the Previous/Next button go?

The previous and next buttons only appear when there are more messages than those currently displayed on the screen.

Help with viewing messages

1. Who can view the messages available in the Message Centre?

A business owner and an authorized representative who is authorized for all accounts can view the messages for all accounts.

An authorized representative who has authorization for specific types of accounts can only view messages related to those accounts.

2. What information will be displayed?

The following information will be displayed for each message:

  • A subject line containing a brief description of the message purpose
  • Date sent
  • Expiry date
  • Message content

If there is no expiry date, the message will be displayed with 9999-12-31 as the expiry date.

3. Can a message be saved?

No. A message will be held in the message centre until it expires or is deleted.

4. Can a message be deleted?

Yes. The message can be deleted by selecting the Delete button. The Message Centre will display with the message removed from the list.

5. Where did the Previous/Next button go?

The previous and next buttons only appear when there are messages to be viewed before or after the message that is displayed.
