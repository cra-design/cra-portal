In your CRA account (using NETFILE) – File your GST/HST return

You can file your GST/HST return and eligible rebates directly with the CRA by signing into your CRA account (My Business Account or Represent a Client). Your return will be sent using the GST/HST NETFILE service.

What you can and cannot do

Most GST/HST registrants can file a regular return using this method.

You can:

  • Meet electronic filing requirements
  • Get a confirmation number right away
  • File without a GST/HST access code
  • Authorize a representative to file on your behalf
  • File most elections electronically
  • File a combined Form RC7200, GST/HST and QST return for selected listed financial institutions
  • File a return for a non-resident
  • File a simplified GST/HST return for a non-resident digital-economy business
  • Include schedules
  • File Form GST106, Information on Claims Paid or Credited for Foreign Conventions
  • File certain rebates
    Rebates you can claim when filing your GST/HST return through your account
    • Form GST66, Application for GST/HST Public Service Bodies' Rebate and Self-Government Refund
    • Form GST189, General GST/HST rebate application, with reason codes 1A, 1C, 5, 7, 8, 9, 12, 13, 16, 20, 23, 24, or 25
    • Form GST190, New Housing Rebate Application for Houses Purchased from a Builder
    • Form GST524, New Residential Rental Property Rebate Application

You cannot:

  • File a return for an account administered by Revenu Québec
  • File rebates not listed above (refer to Claim a GST/HST rebate)

File electronically and send forms by mail

If you want to file using your CRA account but need to file an additional form or rebate claim that can't be submitted electronically, include the amount on your electronic return and mail the form to the applicable tax centre.

Make sure the tax centre receives the form on or before the due date of your return.

If you cannot file in a CRA account

Choose another filing method that works for you.

Before you start

If you plan to file in your CRA account, make sure you:

  • Are able to sign in to your CRA account
  • Know what you can and cannot file using this method
  • Have your Business Number (BN), reporting period "from" and "to" dates, and the rest of the information you need to complete your return
  • Read the CRA's statements on security, privacy and non-responsibility for GST/HST electronic services
  • Are aware of the hours of service for electronic GST/HST filing
  • Have the necessary web browser and browser settings

How to file

  1. Sign in to your CRA account
  2. Select your Business account or your Representative account
  3. Select GST/HST and your RT number
  4. On the GST/HST account overview page, select the button File a return
    • Your secure session will expire after 35 minutes of inactivity
    • If you end your filing session before confirming that you want to file your return, the information you entered will not be saved
  5. Choose the type of return you want to file and select Next
  6. Select all of the situations, rebates, and schedules that apply to you and select Next to continue
    • If you have no amounts to report you can file your return faster by selecting I am filing a nil return (all fields are $0)
  7. Follow the instructions to enter your return amounts and select Next when you are finished entering your information
  8. Review your return to make sure it is correct and complete
    • If something is incorrect, select Edit next to an item you want to change or select Previous to go back and make the corrections
  9. When your return is correct and complete, check the Certification box and then select Submit to file your return
  10. After you successfully submit your return you will receive a 6-digit confirmation number
    • Your confirmation number means we have accepted your tax return for processing
    • Keep this confirmation number for your records
  11. If you have an amount owing or are expecting a refund:
    • If you have an amount owing, Proceed to pay (in My Business Account only) or find out how to make a payment to the CRA
    • If you are expecting a refund, speed up your deposit by setting up direct deposit

Filing a GST/HST return for a non-resident in your CRA account

You must:

  • Have a Social Insurance Number (SIN) or have an authorized representative in Canada
  • Complete all GST/HST amounts in Canadian dollars, unless you are filing a simplified GST/HST return as a digital economy business and you have been approved to file in other qualifying foreign currencies
  • Remit Canadian funds using a participating financial institution in Canada
